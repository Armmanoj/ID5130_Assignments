To build, give build.sh execute permissions and run it-
chmod 777 build.sh
./build.sh

The output png files include-
"Q2a.png" the output plotting numerical and analytical solution for 21 points
"Q2bn_100.png" for plotting the numerical and analytical solution for n = 100
"Q2btiming.png" plots the code timings
Code to plot timings for n=1000 have been commented out in build.sh
